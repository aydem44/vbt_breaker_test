# 🤖 Bybit Trading Bot

Торговый робот на основе стратегии пробоя диапазона (Range Breakout), реализованный на Python с использованием библиотек `pybit` и `pandas`.

## 📊 Стратегия

Основана на бэктестировании стратегии "Breaker Indicator":
- Определяется узкий диапазон за последние N дней
- Сигнал на вход — пробой диапазона вверх/вниз
- Стоп-лосс устанавливается на основание сигнальной свечи
- Тейк-профит с соотношением риск/прибыль 1:1

### Лучшие параметры (найдены в бэктесте)
| Параметр | Значение |
|----------|----------|
| `volatility` | 0.05 (5%) |
| `range_days` | 3 дня |
| `reward_risk` | 1 |
| `lookback` | 90 дней |
